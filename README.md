# ソーラーシェアリング設計シミュレーター

農地・法面・空き地などへの太陽光パネル設置前に、パネル配置と日影の動きを直感的にシミュレーションできる Web アプリです。

> **注意**: 本ツールは初期検討・設計案比較のための簡易シミュレーターです。正式な発電量計算・営農影響評価・構造計算を代替するものではありません。

## 機能（MVP）

- 地図表示（OpenStreetMap ベース）
- 住所または緯度経度で設置地点を指定
- 藤棚型パネルの配置シミュレーション
  - パネルサイズ・高さ・傾斜角・列数・行数・間隔を設定
- 太陽位置リアルタイム計算（SunCalc 使用）
- パネルから地表面に落ちる影を地図上に可視化
- 24時間スライダーで影の動きをアニメーション確認
- 春分・夏至・秋分・冬至の代表日切り替え
- 設計条件の JSON エクスポート

## セットアップ

```bash
# リポジトリをクローン
git clone https://github.com/<your-username>/solar-nm.git
cd solar-nm/client

# 依存パッケージをインストール
npm install
```

## 起動方法

```bash
cd client
npm run dev
```

ブラウザで http://localhost:5173 を開いてください。

## ビルド

```bash
cd client
npm run build
```

`client/dist/` に静的ファイルが生成されます。

## デプロイ

### GitHub Pages

リポジトリの Settings → Pages → Source を「GitHub Actions」に設定してください。  
`main` ブランチへのプッシュで自動的にビルド・デプロイされます。

サブパスが必要な場合は `.github/workflows/deploy.yml` の `VITE_BASE_URL` を調整してください。

### Vercel / Netlify

- Root Directory: `client`
- Build Command: `npm run build`
- Output Directory: `dist`

## 使い方

1. **設置地点を指定** — 住所検索または緯度経度を入力
2. **パネル条件を設定** — 右側パネルでサイズ・高さ・傾斜角などを入力
3. **日時を選択** — 季節プリセットまたは日付を選び、スライダーで時刻を調整
4. **影の動きを確認** — ▶ ボタンで 24 時間アニメーションを再生
5. **データを保存** — JSON ダウンロードで設計条件をエクスポート

## 影計算の前提

- 地形の起伏は未考慮（平坦地想定）
- パネルは剛体平板として扱い、4頂点を独立に地表面へ投影
- 簡易モデル: `影の長さ = パネル高さ / tan(太陽高度角)`
- 太陽位置: SunCalc ライブラリ（精度は十分な初期検討レベル）
